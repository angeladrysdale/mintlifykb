title: "Getting started"
---
The following steps describe how to create a guaranteed deal. For the current guaranteed deals open beta program, only video supply is supported and you must be using one of the following ad servers:

- FreeWheel
- Google Ad Manager
- SpringServe

Use must also use one of the following DSPs:

- The Trade Desk
- Display & Video 360
- Yahoo
- Basis
- StackAdapt
- Beeswax

Contact your Index Representative to join the beta program.

1. To submit API requests, you will first need to establish authentication using one of the following options:
   - To use the same credentials that you use to log in to the Index UI, use the [Generate user account token](ref:loginpost) route.
   - To use a service account, use the [Generate service account token](ref:tokenpost) route.
2. Create the deal using the [Create direct deal](ref:publisherdealsput) route in the Deals API, with the following settings:
   1. Set the `auctionType` field to `fixed`.
   2. In the `directConfigurations` object, specify either a `pgGoalImpression` or a `pgGoalSpend` value.
   3. Set the`programmaticGuaranteed` field to `true`.
   4. Set the `openMarket` field to `false`.
   5. Set the `priority` field to `10`.
   6. Set the `humanVerification` field to `false`.
3. Make sure to also include the following required body parameters in the request:
   - `dspID`
   - `endDate`
   - `legacyAccountID`
   - `name`
   - `rate`
   - `startDate`  
     Refer to the  [Create direct deal](ref:publisherdealsput) route description for more information about these parameters.
